# Kategori-Sulit-Soal-1
1.Program meminta input jumlah siswa (jumlahSiswa).
2.Menggunakan for loop untuk membaca nilai setiap siswa.
3.Nilai setiap siswa dijumlahkan ke variabel totalNilai.
4.Setelah semua nilai dimasukkan, dihitung rata-rata dengan totalNilai / jumlahSiswa.
5.Rata-rata ditampilkan ke layar.
